📖 Project Title

Nexus – Business Collaboration Platform

📌 Project Summary

Nexus is a modern frontend-based business collaboration platform designed to connect entrepreneurs and investors. It simulates a real-world startup ecosystem where users can:

Create and manage profiles
Discover startups and investors
Send collaboration requests
Schedule meetings
Manage documents
Simulate payments and wallet system
Communicate through messaging UI

The system is fully role-based (Investor / Entrepreneur) and provides a complete UI/UX workflow for startup funding interactions.
